# QuickSay Sound Files

This folder should contain the following sound files:

1. **start.wav** - Played when recording starts (high beep)
2. **stop.wav** - Played when recording stops (low beep)
3. **error.wav** - Played on errors

## Option 1: Use System Beeps (Automatic)
The script will use Windows system beeps if no sound files are present:
- Start: 1000 Hz beep
- Stop: 600 Hz beep
- Error: 300 Hz beep

## Option 2: Download Free Sounds
You can download free notification sounds from:
- https://freesound.org
- https://mixkit.co/free-sound-effects/notification/

Recommended: Short, subtle sounds (100-300ms)
