QuickSay uses the Groq API to convert your voice to text in real-time. You'll need a free API key to get started — it takes about 1 minute to set up.

An API key is a unique password that connects QuickSay to Groq's speech-to-text service. It's free, takes 1 minute, and your audio is never stored.

  1. Go to console.groq.com/keys and sign up (Google, GitHub, or email)
  2. Click "Create API Key" and name it anything (e.g., "QuickSay")
  3. Copy the key (starts with gsk_...) and paste it below
