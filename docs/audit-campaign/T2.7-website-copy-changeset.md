# T2.7 → M.3 Website Copy Changeset (READY TO APPLY)

**Status:** Staged, NOT applied. Apply only if the M.3 decision is to **ship** opt-in telemetry.
**Scope:** Desktop-app telemetry claims only. Website Plausible analytics copy is unaffected and left as-is.
**Do not apply early:** Applying before T2.7 actually ships would make the site claim telemetry exists when it doesn't — inaccurate in the opposite direction. These edits land *with* the telemetry release.

---

## Approved framing

- **Off by default** is the headline fact. Lead with it.
- Keep the claims that remain 100% true: zero screen capture, nothing stored on our servers, no PII/transcripts/audio.
- Replace absolute telemetry denials ("zero telemetry" / "zero tracking" / "collects nothing") with the honest opt-in version.
- Canonical long form:
  > "Privacy-first: anonymous analytics are off by default. You can optionally turn on usage stats (6 aggregate events — no transcripts, no audio, no personal data) to help improve QuickSay. QuickSay never captures your screen and stores nothing on our servers."
- Canonical short form: **"Anonymous analytics, off by default."**
- Table-cell form: **"Opt-in"** (was "None" / "Zero").
