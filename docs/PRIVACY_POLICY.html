QuickSay Privacy Policy

Last Updated: February 2026 | Version: 1.0

Overview

QuickSay is a voice-to-text application that runs locally on your Windows computer. We are committed to protecting your privacy and being transparent about how your data is handled.

Key Point: QuickSay developers do not operate any servers and do not collect any of your data. All data is stored locally on your computer or processed through third-party services (Groq API) using your own API key.

Data Collection Summary

Data Type Stored Locally Sent to Cloud Purpose
Audio recordings Temporary only Yes (to Groq) Speech-to-text conversion
Transcription text Yes (if History enabled) No History feature
Groq API key Yes (encrypted) Yes (for authentication) API access
Usage statistics Yes No Personal statistics display
Custom dictionary Yes No Spelling corrections

Detailed Data Handling

1. Audio Recordings

  • Capture: When you activate QuickSay, audio is recorded from your selected microphone
  • Local Storage: Audio is temporarily saved during processing, then deleted
  • Cloud Transmission: Audio is sent to Groq's servers for transcription via their API
  • Retention: Groq's data retention is governed by their privacy policy

2. Groq API Integration

  • Your API Key: Stored locally on your computer, encrypted using Windows Data Protection API (DPAPI)
  • Transmission: Your API key is sent with each transcription request to authenticate with Groq
  • Groq's Processing: Audio is processed by Groq's Whisper model. Refer to Groq's Privacy Policy for their data handling practices

3. Local Data Storage

All of the following data is stored ONLY on your local computer:

  • Transcription History: data\history.json
  • Usage Statistics: data\statistics.json
  • Custom Dictionary: dictionary.json
  • Settings: config.json

4. What We Do NOT Collect

  • We do not operate any servers
  • We do not collect analytics or telemetry
  • We do not track your usage patterns
  • We do not have access to your transcriptions
  • We do not store your API key anywhere except on your computer

Third-Party Services

Groq API

QuickSay uses Groq's API for speech-to-text conversion. When you use QuickSay:

  • Your audio is sent to Groq's servers
  • Groq processes the audio and returns text
  • Groq's handling of your data is governed by their privacy policy

You are responsible for:

  • Reviewing and accepting Groq's terms of service
  • Managing your own Groq API key
  • Understanding Groq's data retention policies

Data Security

Encryption

  • Your Groq API key is encrypted using Windows DPAPI (Data Protection API)
  • DPAPI encryption is tied to your Windows user account
  • The encrypted key can only be decrypted on your computer by your user account

Local Storage

  • All local data is stored in your user profile directory
  • Data is protected by your Windows user account permissions
  • No data is transmitted to QuickSay developers

Your Rights and Controls

You Can:

  • View all stored data in the Settings app (History, Statistics tabs)
  • Delete your transcription history at any time
  • Export your custom dictionary
  • Uninstall QuickSay completely, removing all data

Data Deletion

  • Transcription History: Settings > History > Clear History
  • Complete Removal: Uninstall QuickSay to remove all local data

Children's Privacy

QuickSay is not intended for use by children under 13. We do not knowingly collect data from children.

Changes to This Policy

We may update this Privacy Policy periodically. Changes will be indicated by updating the "Last Updated" date. Continued use of QuickSay after changes constitutes acceptance of the updated policy.
